### Задача 3.1: "Секретный код от сейфа"
В системе сохранился фрагмент лога, в котором зашифрован код доступа. Код находится **после** слова `CODE:` и состоит ровно из 4 символов. Напишите программу, которая находит индекс слова `CODE:`, вычисляет место, где начинается сам код, и извлекает его.

**Пример данных:**
```python
log_data = 'Системное сообщение: Ошибка доступа. CODE:5849 завершение сессии'
# Ваш код должен вывести: 5849
```

<details>
<summary>Подсказка</summary>
Сначала найдите индекс начала слова `CODE:` с помощью `find`. Затем к этому индексу прибавьте длину самого слова `CODE:` (5 символов), чтобы попасть на начало цифр. После этого сделайте срез.
</details>

### Задача 3.2: "Генератор уникального ID"
Клиент оставил отзыв. Нам нужно создать уникальный ID для этого клиента в базе данных.
Правило формирования ID:
1. Взять **последние 3 символа** отзыва.
2. Перевернуть их задом наперед (например, из `'кот'` получить `'ток'`).
3. Добавить в начало общую длину отзыва.

**Пример данных:**
```python
feedback = 'Отличный номер, рекомендую!'
# Длина строки 28 символов. Последние 3 символа: 'ю!'.
# Перевернутые: '!ю'.
# Итоговый ID: '28!ю'
```

<details>
<summary>Подсказка</summary>
Используйте `len(feedback)` для числа. Для получения последних символов используйте отрицательный индекс среза `[-3:]`. Чтобы перевернуть строку без новых функций, вам придется обратиться к каждому символу по отдельности и сложить их в обратном порядке.
</details>

### Задача 3.3: "Анализ цен в чеке"
В тексте чека скрыта информация о самой дорогой позиции. Строка выглядит так: `Price:XXXX`, где `XXXX` — это число. Напишите программу, которая находит слово `Price:`, извлекает следующие за ним 4 символа (это будет строка с числом), преобразует её в число и увеличивает на 10% (умножает на 1.1).

**Пример данных:**
```python
receipt = 'Товар: Пицца, Price:0500, Скидка: 10%'
# Нужно извлечь '0500', превратить в число 500, посчитать 550 и вывести результат.
```

<details>
<summary>Подсказка</summary>
1. Найдите индекс `'Price:'` через `find`.
2. Сделайте срез, начиная с `index + 6` (так как в слове Price: 6 букв) и заканчивая `index + 10` (4 цифры).
3. Не забудьте применить `int()` к результату среза, прежде чем умножать.
</details>

### Задача 3.4: "Поиск скрытого ключа"
Пользователь ввел строку, в которой может быть скрыт ключевой код. Ключ всегда пишется слово `KEY`. Но пользователь мог ввести его маленькими буквами. Напишите программу, которая приводит строку к нижнему регистру, ищет слово `'key'` и, если находит его, выводит сообщение: "Ключ найден на позиции X", где X — это индекс начала слова. Если слово не найдено, `find` вернет -1, и это тоже нужно вывести.

**Пример данных:**
```python
user_input = 'Секретный пароль: KEY123'
# Программа должна вывести: Ключ найден на позиции 18
```

<details>
<summary>Подсказка</summary>
Используйте метод `.lower()` для переменной перед поиском. Сохраните результат `find` в переменную и используйте её в `print`.
</details>

### Задача 3.5: "Конструктор email-адреса"
Из имени и фамилии сотрудника нужно создать корпоративную почту.
Формат: `первая_буква_имени + вся_фамилия + @company.com`.
Обратите внимание: буква должна быть маленькой, даже если имя введено с большой.

**Пример данных:**
```python
name = 'Ivan'
surname = 'Petrov'
# Результат должен быть: ipetrov@company.com
```

<details>
<summary>Подсказка</summary>
1. Возьмите `name[0]` и приведите её к нижнему регистру `.lower()`.
2. Приведите фамилию к нижнему регистру целиком.
3. Используйте конкатенацию (сложение) строк, чтобы собрать все части воедино.
</details>

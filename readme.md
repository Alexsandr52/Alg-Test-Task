Вот 5 задач, разработанных специально для студентов. Они охватывают работу с переменными, строками, функциями `find`, `lower`, `int` и требуют использования вложенных циклов и условных конструкций.

**Важное условие:** В решениях запрещено использовать логический тип данных (`bool`, `True`/`False`). Для проверки условий используйте сравнение индексов (например, `index != -1`), целочисленные флаги (0 или 1) или математические операции.

### Задача 1: «Редактор конфиденциальных данных»
**Суть:** Программа должна находить в тексте конкретное "запрещенное" слово и заменять его на звездочки, независимо от регистра букв.

**Входные данные:**
1. Строка с текстом (например: *"У меня есть iPhone, но я хочу iphone"*).
2. Запрещенное слово (например: *"iphone"*).

**Задание:**
Напишите программу, которая заменяет все вхождения запрещенного слова в тексте на символ `*`. Количество звездочек должно совпадать с длиной слова.
*   Необходимо использовать функцию `lower()` для сравнения слов без учета регистра.
*   Используйте `find()` в  для поиска всех вхождений.
*   Не используйте готовый метод `replace()` — реализуйте замену вручную через срезы (slices) и конкатенацию.

**Пример работы:**
> Ввод: "Hello World", "world"
> Вывод: "Hello *****"

### Задача 4: «Парсер логов»
**Суть:** Проанализировать строку лога, извлечь из неё код ошибки и определить уровень критичности.

**Входные данные:**
Строка в формате: `LEVEL: Code=XXX`, где `XXX` — число (например: *"Error: Code=500"* или *"Info: Code=200"*).

**Задание:**
1. Найдите индекс символа `'='` с помощью `find()`.
2. Извлеките подстроку после `'='` и преобразуйте её в целое число (`int()`).
рат.

**Пример работы:**
> Ввод: "Error: Code=504"
> Вывод: "ошибка: code=504" (или похожее сообщение с lower())
> Ввод: "Success: Code=201"
> Вывод: 40401

### Задача 5: «Конструктор ID»
**Суть:** Сгенерировать уникальный идентификатор на основе частей строки и их длин.

**Входные данные:**
Строка, содержащая два слова, разделенных дефисом (например: *"admin-root"*).

**Задание:**
1. Используя `find('-')`, разделите строку на две части: `part1` и `part2`.
2. Преобразуйте `part1` в нижний регистр (`lower()`).
3. Определите длину второй части (`part2`).
4. Сформируйте итоговую строку в формате: `part1` + длина `part2` + первая буква `part2`.
*   Задача требует аккуратной работы с индексами и срезами.
*   Все проверки наличия дефиса должны осуществляться через сравнение результата `find` с `-1`.

**Пример работы:**
> Ввод: "Super-User"
> Вывод: "super4U" (super + длина слова "User" (4) + первая буква "U")

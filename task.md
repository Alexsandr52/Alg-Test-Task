
# Дополнительные задания: Вложенные вызовы функций

В этих задачах важно не создавать лишние переменные, а сразу применять функции к вводу пользователя.

### Задача 4.1: "Стандартизация команд"
Чат-бот принимает команды от пользователя, но он может написать "Пуск", "ПУСК" или "пуск". Чтобы программа работала правильно, нужно сразу привести ввод к нижнему регистру.
Напишите программу, которая запрашивает команду и сразу (в одной строке) переводит её в нижний регистр, а затем выводит.

**Пример работы:**
```python
# Пользователь вводит: НАЧАТЬ
# Программа выводит: начать
```

<details>
<summary>Подсказка</summary>
Используйте конструкцию: `переменная = input(...).lower()`
</details>

### Задача 4.2: "Калькулятор возраста"
Пользователь вводит свой возраст. Программа должна сразу преобразовать эту строку в число и посчитать, через сколько лет ему исполнится 100 лет.

**Пример работы:**
```python
# Пользователь вводит: 25
# Программа выводит: До 100 лет осталось: 75
```

<details>
<summary>Подсказка</summary>
Считайте возраст так: `age = int(input('Введите возраст: '))`. Затем выполните вычитание `100 - age`.
</details>

### Задача 4.3: "Проверка длины пароля"
Нам нужно узнать длину пароля, который вводит пользователь, не сохраняя сам пароль в отдельную переменную.
Напишите программу, которая запрашивает пароль и сразу передает его в функцию `len()` для подсчета символов.

**Пример работы:**
```python
# Пользователь вводит: secret123
# Программа выводит: Длина пароля: 9
```

<details>
<summary>Подсказка</summary>
Вызов функции можно вложить внутрь другой: `len(input(...))`.
</details>

### Задача 4.4: "Сумма покупок"
Покупатель вводит стоимость товара и количество единиц. Оба значения нужно сразу преобразовать в числа и посчитать общую сумму.

**Пример работы:**
```python
# Пользователь вводит стоимость: 150
# Пользователь вводит количество: 3
# Программа выводит: Итого: 450
```

<details>
<summary>Подсказка</summary>
Используйте вложенный `int()` для обоих input: `price = int(input(...))` и `count = int(input(...))`. Затем перемножьте их.
</details>

### Задача 4.5: "Мгновенный поиск" (Повышенная сложность)
Это задание объединяет вложенность и методы строк.
Пользователь вводит текст. Программа должна одной строкой кода сделать следующее:
1. Прочитать текст.
2. Привести его к нижнему регистру.
3. Найти в нем слово `python`.
Вывести индекс найденного слова.

**Пример работы:**
```python
# Пользователь вводит: Я изучаю Python и мне нравится
# Программа выводит: 11
```

<details>
<summary>Подсказка</summary>
Методы можно вызывать цепочкой. Сначала `.lower()` приводит текст к нижнему регистру, а к результату сразу применяется `.find('python')`.
Код будет выглядеть так: `index = input(...).lower().find(...)`
</details>
